What is userSchema.methods?
It's used to add custom functions (methods) to documents (each user).

These methods can use this to access the document's data.


methods = functions you can call on a document like user.sayHello().

userSchema.methods.sayHello = function() {
  return `Hi, I'm ${this.name}`;
};


const User = mongoose.model('User', userSchema);

User.findOne().then(user => {
  console.log(user.sayHello());  // Output: Hi, I'm Lina
});
