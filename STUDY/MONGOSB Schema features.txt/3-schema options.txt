3. Schema Options

Modify schema behavior.

timestamps: auto adds createdAt and updatedAt

versionKey: disables __v

collection: custom collection name

const orderSchema = new mongoose.Schema({
  product: String
}, {
  timestamps: true,
  versionKey: false,
  collection: 'orders'
});


Mongoose will automatically create these two fields for each document:

createdAt
updatedAt

const userSchema = new mongoose.Schema({
  name: String,
  email: String
}, {
  timestamps: true
});


res.render('profile', { user });
<p>Created At: <%= user.createdAt.toLocaleString() %></p>
<p>Last Updated: <%= user.updatedAt.toLocaleString() %></p>
